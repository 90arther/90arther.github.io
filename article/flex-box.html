用box-flex创造三列布局或多列

首先查询一下flexble box layout module浏览器兼容性

请戳这个地址：http://caniuse.com/#search=flexbox

我们发现，在IE下，8/9是不支持的。10需要加前缀。到11才完全支持。

在其它浏览器下，chrome要到31版本才支持，firefox要到28才完全支持。可以看出PC站，兼容性还是比较差的。但是在主流手机端的浏览器支持情况还是比较理想的。

所以在目前情况下，flexible box layout module 比较适合开发手机端的产品。

    .fath{
        display: -webkit-box;
        display: -moz-box;
        display: -ms-flexbox;
        display: box;
    }
    .fath > div{ outline: 1px solid red;}
    .bisection { 
        -moz-box-flex: 1;
        -webkit-box-flex: 1;
        -ms-flex: 1;
        flex: 1;
    }
